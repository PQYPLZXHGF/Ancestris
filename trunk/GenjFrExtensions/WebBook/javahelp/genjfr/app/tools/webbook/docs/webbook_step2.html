Assistant - Panneau "Individus"


Ce panneau permet de personnaliser plusieurs éléments liés aux individus :
  1. Le choix de l'individu de-cujus (individu principal). C'est cet individu qui apparaîtra sur la page d'accueil, et c'est à partir de lui que seront indiqués les numéros sosa et le rapport d'ancêtres.
  2. La façon d'afficher les individus dont le nom est inconnu, dans toutes les sections du livre Web.
  3. Les informations à afficher ou pas pour chaque individu de la section détaillée, comme sa famille, ses enfants, ses parrains et marraines, les notes rattachées, etc.
  4. L'affichage ou pas du numéro d'identification Ancestris après le nom de chaque individu, dans toute les sections du Livre.
  5. Les critères de définition des données privées. Le bouton affiché permet de modifier les préférences de données privées d'Ancestris comme si vous y accédiez depuis le menu "Préférences".
